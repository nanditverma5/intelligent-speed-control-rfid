====================================================
      RFID-BASED INTELLIGENT SPEED CONTROL SYSTEM
====================================================

                 ┌───────────────────────────┐
                 │        ARDUINO UNO        │
                 │       (Microcontroller)   │
                 └──────────────┬────────────┘
                                │
                ┌───────────────┴────────────────┐
                │                                │
      ┌─────────┴──────────┐           ┌─────────┴──────────┐
      │   RFID RC522       │           │   MOTOR DRIVER L293D │
      │   (Reader Module)  │           │                      │
      └─────────┬──────────┘           └─────────┬──────────┘
                │                                │
     SDA  → Pin 10                                │   Motor Output → DC Motor
     SCK  → Pin 13                                │   (Speed controlled)
     MOSI → Pin 11                                │
     MISO → Pin 12                                │
     RST  → Pin 9                                 │
     VCC  → 3.3V                                  │
     GND  → GND                                   │

====================================================
     MOTOR SPEED CONTROL (PWM)
====================================================

Arduino Pin 5 (PWM) ─────────► L293D ENA pin  
L293D OUT1/OUT2 ─────────────► DC Motor

====================================================
     POWER SUPPLY
====================================================

RFID Module ─────► 3.3V  
Arduino UNO ─────► USB/5V  
Motor Driver ────► External 6–12V supply
